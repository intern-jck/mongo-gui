# mongo-gui

## Starter template for building a MongoDB GUI


To start development, run the follwing:

```bash
npm install
```

Start mongod

```
sudo systemctl start mongod
sudo mongosh
use mongogui
```

Start server, then client

```
npm run server-dev
npm run client-dev
```

Enjoy!
